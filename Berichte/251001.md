# Grundlagen

Shell: ist ein Programm das ein textorientierte Schnittstelle zum Betriebssystem zur Verfügung 

Promt:

```sh
~/workspace$
```

Einfacher Befehl:

```sh
$ date
```

Jeder Befehl hat Optionen über die das Verhalten gesteuert werden kann.

```sh
$ date -I
```

## Optionen:
- short-hand = Buchstabe
- long-hand = Wort

```sh
$date --iso-8601
```

Quellen zu Informationen über einen Befehl:

- häufig: Option `--help`
- man pages (Manual = Handbuch) `$ man date`, nicht in Replit aber in gänigen Linux Distributionen
- Internet
- KI


## Argumente

Daten mit denen ein Befehl arbeitet.


```sh
$echo Hallo Welt
```

`Hallo Welt` ist hier das Argument.
mit Optionen
```sh
$echo -n Hallo Welt
```
## Tastenkürzel

- Pfeil nach Oben /unten: Historie
- Tabulator: automatisch File Vervollständigung.
- Strg-A: Anfang der Zeile (Strg ... Ende)
- Strg-C: Abbruch des aktuellen Kommandos

## File Befehle  

cd ändert arbeits verzeichniss, mkdir erstellt einen ordner, ls ließt inhalt von arbeitsverzeichniss aus, touch legt neue datei an, pwd, rm löscht datei
workind directora (Arbeitsverzeichnis)

Unterschied absoluter und relativer Pfad:Ein absoluter Pfad gibt die vollständige Position einer Datei im Dateisystem vom Stammverzeichnis an, z. B. C:\Benutzer\Dokumente\Datei.txt. Ein relativer Pfad hingegen gibt die Position einer Datei im Verhältnis zum aktuellen Verzeichnis an, z. B. Bilder/Urlaub.jpg(google ai Promt: unterschied relativer un Absoluter Pfad)

Bedetung von cd(ohne argument):wechselt in der Linux-Shell direkt in Ihr Home-Verzeichnis.(google ai Promt: Was bedeutet cd(ohne argument) in Linux)

Bedeutung von ~:  Es repräsentiert typischerweise das Heimatverzeichnis des aktuellen Benutzers, also das Stammverzeichnis des Benutzers auf dem System.(google ai Promt: Was bedeutet ~ in Linux )
