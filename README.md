## Tipps zur Shell-Nutzung
Wenn du das Zeichen `/` (Slash) nicht eingeben kannst:
- Auf einer **deutschen Tastatur** liegt `/` auf **Umschalt (Shift) + 7**.
- Wenn die Tastatur auf **Englisch (US)** eingestellt ist, liegt `/` auf der Taste links neben der rechten Umschalt-Taste (wo im Deutschen `-` und `_` sind).
- Du kannst Pfade auch mit der **Tab-Taste** vervollständigen lassen, dann schreibt die Shell den `/` oft automatisch.
